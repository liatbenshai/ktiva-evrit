# ⚡ עדכון מהיר - מה לעשות עכשיו

## שלב 1: קחי את ה-URL הזה

**החלפי `[YOUR-PASSWORD]` בסיסמה האמיתית שלך:**

```
postgresql://postgres.kpplrkgkhkhgrnjwgfpb:[YOUR-PASSWORD]@aws-0-eu-west-1.pooler.supabase.com:6543/postgres
```

## שלב 2: עדכני את .env.local

1. פתחי את `.env.local` (אם הוא לא קיים, צרי אותו)
2. הוסיפי או עדכני את השורה:
   ```env
   DATABASE_URL=postgresql://postgres.kpplrkgkhkhgrnjwgfpb:YOUR-PASSWORD@aws-0-eu-west-1.pooler.supabase.com:6543/postgres
   ```
3. שמרי את הקובץ

## שלב 3: בדקי

```bash
npm run db:check
```

אמור לראות:
```
✅ Connection Pooling (מומלץ ל-Vercel)
📍 פורט: 6543
✅ חיבור הצליח!
```

## אם זה לא עובד

אם את רואה שגיאה, נסי את האפשרויות האחרות מקובץ `ה-URL_הנכון_לך.md`:
- אפשרות 2: `eu-central-1`
- אפשרות 3: `us-east-1`
- אפשרות 4: `us-west-1`

או בדקי מה ה-Region ב-Supabase Dashboard → Settings → General
