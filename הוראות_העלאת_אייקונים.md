# 📱 הוראות העלאת אייקונים ל-PWA

## איפה להעלות?

העלי את הקבצים לתיקייה: **`public/`**

## איזה קבצים צריך?

1. **`icon-192.png`** - בגודל 192x192 פיקסלים
2. **`icon-512.png`** - בגודל 512x512 פיקסלים

## איך להעלות?

### דרך Cursor/VS Code:
1. פתחי את התיקייה `public/` בפרויקט
2. גררי את הקבצים לתוך התיקייה
3. ודאי שהשמות הם בדיוק:
   - `icon-192.png`
   - `icon-512.png`

### דרך GitHub (אם יש לך גישה):
1. לכי ל-`public/` ב-GitHub
2. לחצי על "Add file" → "Upload files"
3. העלי את הקבצים
4. ודאי שהשמות נכונים

## מה צריך להיות בתיקייה public/?

```
public/
  ├── icon-192.png  ← כאן
  ├── icon-512.png  ← כאן
  ├── manifest.json
  ├── sw.js
  └── favicon.ico
```

## אחרי ההעלאה:

1. הקבצים יופיעו אוטומטית ב-`/icon-192.png` ו-`/icon-512.png`
2. ה-PWA יזהה אותם אוטומטית
3. כשמתקינים את האפליקציה, האייקונים יופיעו במסך הבית

## בדיקה:

אחרי שהעלית, בדקי שהקבצים נגישים ב:
- `https://your-domain.com/icon-192.png`
- `https://your-domain.com/icon-512.png`

## הערה:

אם קלאוד נתן לך קבצים בשמות אחרים, פשוט שנה את השמות ל:
- `icon-192.png` (אם זה האייקון הקטן)
- `icon-512.png` (אם זה האייקון הגדול)
